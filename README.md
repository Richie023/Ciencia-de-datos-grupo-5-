# 📊 Ciencia de Datos - Grupo 5

Repositorio oficial del curso de **Ciencia de Datos** impartido por **PROCOMER**, donde se almacenan los apuntes, ejercicios, proyectos y recursos desarrollados a lo largo del programa.

---

## 🎯 Objetivo

Organizar y documentar el aprendizaje obtenido durante el curso, reuniendo en un solo lugar el material teórico y práctico relacionado con Ciencia de Datos.

---

## 📚 Temas del curso

- Fundamentos de Ciencia de Datos
- Manejo y Preparación de Datos
- Análisis Exploratorio de Datos (EDA)
- Visualización de Datos y Business Intelligence (BI)
- Introducción a Machine Learning
- Proyecto Integrador de Análisis de Datos
- Habilidades Socioemocionales para el Analista de Datos

---



## 📖 Contenido

Este repositorio incluirá:

- 📝 Apuntes de cada clase
- 💻 Ejercicios prácticos
- 📊 Análisis de datos
- 📈 Visualizaciones

- 📂 Datasets utilizados durante el curso

## 🛠️ Tecnologías y herramientas

- Python


## 🚀 Objetivos de aprendizaje

Al finalizar el curso se espera desarrollar habilidades para:

- Comprender los fundamentos de la Ciencia de Datos.
- Preparar y limpiar conjuntos de datos.
- Realizar análisis exploratorio (EDA).
- Crear visualizaciones para apoyar la toma de decisiones.
- Aplicar modelos básicos de Machine Learning.
- Desarrollar proyectos de análisis de datos utilizando buenas prácticas.
